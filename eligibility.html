Validity Parameters

Criteria for Project Acceptance

Team Validity

Team Name

Must match the project name exactly.

Team Size

1 to 6 eligible participants per team.

Leadership (C/AC)

Captain and Assistant must be designated for communication.

Supervision

Required (18+) if all members are minors.

Representation

Determined by the majority of teammates.

Project Validity

Project Name

Must match the team name exactly.

Project Type

Classified as Architecture (ARC) or Development (DEV).

Representation

Must align with Team Representation.

Name Validity

Meaning & IP

Names must be descriptive and respect existing intellectual property.

Harmonious

No offensive, political, or unethical content allowed.

Length

3–4 words maximum; taglines are encouraged.

KBI reserves the right to amend or disqualify a team or project if these parameters are not met or if amendments are not accepted.
